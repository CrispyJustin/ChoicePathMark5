# ChoicePath static Vite deployment

This package removes the TanStack Start/TanStack Router runtime and deploys ChoicePath as a plain Vite React single-page app.

Vercel settings:
- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist

Environment variables:
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
